Description

Provides a base class for canvas item shapes, including: MateCanvasBpath, MateCanvasEllipse, MateCanvasRect, and MateCanvasPolygon.

Details

struct MateCanvasShape

struct MateCanvasShape;

This structure should not be accessed directly. Use the object's properties and functions for querying or changing its parameters.

mate_canvas_shape_set_path_def ()

void                mate_canvas_shape_set_path_def     (MateCanvasShape *shape,
                                                         MateCanvasPathDef *def);

This function sets the the MateCanvasPathDef used by the MateCanvasShape. Notice, that it does not request updates, as it is meant to be used from item implementations, from inside update queue.

shape :

 a MateCanvasShape

def :

 a MateCanvasPathDef

mate_canvas_shape_get_path_def ()

MateCanvasPathDef * mate_canvas_shape_get_path_def    (MateCanvasShape *shape);

This function returns the MateCanvasPathDef that the shape currently uses. It adds a reference to the MateCanvasPathDef and returns it, if there is not a MateCanvasPathDef set for the shape it returns NULL.

shape :

 a MateCanvasShape

Returns :

 a MateCanvasPathDef or NULL if none is set for the shape.
