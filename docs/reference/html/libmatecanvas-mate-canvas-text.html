Description

A canvas item for displaying text. See MateCanvasRichText for a more advanced text display and editing canvas item. NB: The MateCanvasText item doesn't scale with the zoom property of the MateCanvas. A zoomable implementation could derive from MateCanvasText and check the zoom property for manual adjustments to the font size.
